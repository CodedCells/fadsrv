Mark Help

What are marks?

Marks are a way of categorising and organising posts.

Akin to favouriting a post on Fur Affinity, but with so many more options.


Generally represented as buttons or icons, they can be used for almost anything.

Types

Marks have a type property, indicating how they can be used.


Multi-Button ("type": "multibutt")

Creates one or more buttons that can be picked between and toggled.

Dropdown List ("type": "list")

Creates a dropdown list of predefined values.

Collection ("type": "collection")

Creates a button which opens a menu of user-created collections.

Int ("type": "int") unused

Creates a numerical input, untested outside of direct use.

Others

These are the only ones implemented in the base code.

I'd like to add user-customisable functionality to this, but it works for all my use cases right now.

Properties

Other properties can be specified for marks, as to how they will behave.

For ("for": "kind")

Semi-required property, defaults to posts.

Default values include:

  • posts
  • posts_unav
  • users
  • postmarks
  • tags
  • folders
  • sets

Any additional marks may also be used, e.g. sets.

Icon ("icon": [x, y])

Optional, defaults to the name of the value or mark, may also default to [9, 9] in some cases.

Refers to a position on this iconsheet, starting at [0, 0] at the top left.

x refers to the horizontal position, increasing going towards the right.

y refers to the vertical position, increasing going towards the bottom.

Values ("values": ["thing1", "thing2"])

Values are used in multi-button and dropdown list mark files.

A list of values, where the user can pick between them or to unset it.

Value Icon ("valueicon": [[4,3], [3,3]])

Value Icons are used aside values, providing a unique icon for each value.

Exclude Marked ("excludeMarked": false)

Defaults true, meaning it'll hide content marked by it

Whether to exclude posts marked by this when docats is true.

Order ("order": 10)

Display order when placed aside other marks.

Disabled ("disabled": true)

Displays button, but makes the button unclickable.

Hidden ("hidden": true)

Do not display at all.

Do Read Count ("doReadCount": true)

Used for collections only, when enabled displays if all posts have been marked as read.

List ("list": true)

USed to tell the system this file is just a list of things separated by lines.

Cannoot be modified at runtime, please set disabled or hidden.
