Grab Post Help

What is grab_post.py?

Grab Post is an automated tool that'll grab the favourites of the user specified in the config.

Designed to be used on your own favourites, but could be set to target someone else's if it's shown.

Setting up

Requirements
  • Python 3.x
  • Requests module
  • Cookies configuration (see Cookies Help)

Try and run grap_post.py.


Configure

After running the program at least once, data/post_grab_cfg.json will be created.

Edit the file and set the username to the target account.

Additionally if you are running FADSRV on another port, change the port number.


Default settings will work with the stock FADSRV configuration.

Usage

The program should run to completion just by opening, downloading all the posts in the user's favourites.


Program Loop Logic

The program will continue to visit the next page until either there are no more pages or it doesn't see any new posts.

This may cause issues if terminated during an initial run.

The program runs at a reduced speed and is intended to run in the background once you've done favouriting, this is to reduce stress on Fur Affinity's servers.

The program downloads the page and then the media for each post, working it's way from newest to oldest.
