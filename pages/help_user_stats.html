User Stats Help

What is user_stats.py?

User Stats is an automated tool that'll collect useful information about users for FADSRV.

Setting up

Requirements
  • Python 3.x
  • Requests module
  • Cookies configuration (see Cookies Help)

Configure

Use the same settings as post_grab.py since i'm lazy.

Usage

Make sure FADSRV is running as this program asks it for a list of users.

Run the program, terminate if you want when it sleeps.


Program Loop Logic

The program runs in three sequential modes


New Users

Collects stats for users it hasn't seen before, considered a priority.

Update Users

Users whose information was collected long ago may be worthwhile updating, starting with the oldest.

Can break out of this loop by holding Ctrl+C, will also stop if Fur Affinity is busy.

Manual Entry

Lets the user enter an account name, or just press enter to quit.
